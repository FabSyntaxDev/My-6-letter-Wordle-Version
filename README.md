# My 6-letter Wordle Version 🧩

Uma versão customizada do famoso jogo Wordle, adaptada para palavras de **6 letras**, oferecendo um desafio extra para os entusiastas de quebra-cabeças linguísticos.

![Repo Size](https://img.shields.io/github/repo-size/FabSyntaxDev/My-6-letter-Wordle-Version)

## 📋 Sobre o Projeto

Este projeto é um clone funcional do Wordle original, mas com uma diferença crucial: em vez das tradicionais 5 letras, o jogador deve adivinhar uma palavra de **6 letras**. O objetivo é descobrir a palavra oculta em até 6 tentativas.

### Como Jogar 🎮
1. Digite uma palavra válida de 6 letras.
2. Após cada tentativa, as cores das peças mudarão para mostrar o quão perto você está da palavra:
   - 🟩 **Verde:** A letra está na palavra e no lugar correto.
   - 🟨 **Amarelo:** A letra está na palavra, mas no lugar errado.
   - ⬛ **Cinza:** A letra não faz parte da palavra.

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

- **HTML5** & **CSS3** (Interface e animações)
- **JavaScript (ES6+)** (Lógica do jogo e manipulação do DOM)
- **Requisição de API externa.
